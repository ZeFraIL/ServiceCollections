# Подробное описание класса TimerService

## 1. Общая информация
* **Имя класса:** `TimerService`
* **Тип:** `Bound Service`
* **Назначение:** Секундомер, к которому можно подключиться для получения данных.

## 6. Объяснение простыми словами
`TimerService` — это **повар** на кухне, который считает секунды, а Activity — это **официант**, который спрашивает: "Сколько времени прошло?".
