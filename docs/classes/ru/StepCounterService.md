# Подробное описание класса StepCounterService

## 1. Общая информация
* **Имя класса:** `StepCounterService`
* **Тип:** `Service`
* **Назначение:** Подсчет шагов через датчики устройства.

## 6. Объяснение простыми словами
`StepCounterService` — это **счетчик пробега** в машине, который считает, сколько вы проехали с начала поездки.
