# Подробное описание класса MainActivity

________________________________________
## 1. Общая информация
*   **Имя класса:** `MainActivity`
*   **Тип:** `Activity`
*   **Назначение:** Является точкой входа в приложение и пультом управления для всех сервисов.
*   **Взаимодействие:** Запускает и останавливает `MusicService`, `SimpleTimerService`, `TimerService`, `StepCounterService` и `CacheCleanerService`.

________________________________________
## 2. Переменные (поля класса)

| Имя | Тип | Назначение |
| :--- | :--- | :--- |
| `context` | `Context` | Контекст приложения. |
| `switchMusic` | `Switch` | Переключатель сервиса. |
| `bStart`, `bStop` | `Button` | Кнопки управления. |

________________________________________
## 3. Методы класса
*   **onCreate()**: Инициализация экрана.
*   **updateTimer()**: Обновление времени в UI.
*   **initComponents()**: Поиск View по ID.

________________________________________
## 8. Объяснение простыми словами
`MainActivity` — это **водитель** автомобиля. У него есть приборная панель (UI), через которую он управляет всеми системами машины (сервисами).
