# Подробное описание класса MusicService

## 1. Общая информация
* **Имя класса:** `MusicService`
* **Тип:** `Service`
* **Назначение:** Фоновое воспроизведение музыки.

## 6. Объяснение простыми словами
`MusicService` — это **уличный музыкант**. Он играет музыку до тех пор, пока его не попросят остановиться.
