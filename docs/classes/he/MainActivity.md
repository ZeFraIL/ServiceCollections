# תיאור מפורט של המחלקה MainActivity

## 1. מידע כללי
* **שם המחלקה:** `MainActivity`
* **סוג:** `Activity`
* **תפקיד:** לוח בקרה לכל השירותים באפליקציה.

## 2. משתנים
| שם | סוג | מטרה |
| :--- | :--- | :--- |
| `context` | `Context` | מידע על סביבת האפליקציה |
| `switchMusic` | `Switch` | שליטה במוזיקה |

## 8. הסבר פשוט
`MainActivity` הוא ה**נהג**. הוא משתמש בלוח המחוונים כדי להפעיל את המערכות השונות של המכונית (השירותים).
